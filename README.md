# 🚀 CareerCraft AI – AI Resume Builder
CareerCraft AI is a modern AI-powered Resume Builder that helps users create professional, ATS-friendly resumes in minutes. It uses AI to generate compelling summaries, improve work experience descriptions, suggest relevant skills, and export resumes as high-quality PDF files.

---

## 📌 Features

- 🤖 AI-powered Resume Content Generation
- 📄 ATS-Friendly Resume Templates
- 📝 Professional Summary Generator
- 💼 Experience Enhancement using AI
- 🎯 Job-Specific Resume Optimization
- 🧠 Skill Recommendations
- 📑 Cover Letter Generator
- 👀 Live Resume Preview
- 📥 Download Resume as PDF
- 🌙 Dark & Light Mode
- 🔐 User Authentication
- ☁️ Cloud Database Support
- 📱 Fully Responsive Design

## ⚙️ Installation
### Clone the Repository
```bash
git clone https://github.com/your-username/careercraft-ai.git
```

### Navigate to the Project
```bash
cd careercraft-ai
```

### Install Frontend Dependencies
```bash
cd client
npm install
```

### Install Backend Dependencies
```bash
cd ../server
npm install
```

### Configure Environment Variables
Create a `.env` file in the server folder.
```env
PORT=5000
MONGODB_URI=your_mongodb_connection
OPENAI_API_KEY=your_api_key
JWT_SECRET=your_secret_key
```

### Start the Backend
```bash
npm run dev
```

### Start the Frontend

```bash
cd client
npm start
```
